WhaleSpotting API:

This is the quickstart guide for the WhaleSpotting public API. This API primarily exposes endpoints aimed at scientists wishing to coduct data analysis on our whale database. It also allows for bulk upload and bulk download of data for analysis/archiving/intense whale spotting.
